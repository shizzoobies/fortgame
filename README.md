# Fortress Defense Web Game - Claude Code Pack

This pack is designed to be dropped into a fresh project and handed to Claude Code as the implementation brief.

Included files:
- `CLAUDE.md` — main build instructions for Claude Code
- `GAME_SPEC.md` — game design and mechanics
- `TASKLIST.md` — implementation order
- `ART_DIRECTION.md` — UI/art generation direction for HTML/CSS/canvas
- `ACCEPTANCE_CHECKLIST.md` — what "done" looks like

Recommended workflow:
1. Open a new folder in Claude Code.
2. Add these files to the root.
3. Tell Claude Code to read `CLAUDE.md` first, then follow the build order in `TASKLIST.md`.
4. Have Claude generate the full playable prototype in plain HTML/CSS/JS.

Target:
A pure web, single-screen fortress defense prototype using HTML5 Canvas and plain JavaScript.
